Everything a terminal agent should do

From reading your codebase to shipping — Calen Code works with you, not against you.

⌘

Terminal-Native

No Electron, no bloat. A single ~26 MB binary with zero runtime dependencies. Linux, macOS, Windows, Termux.

~

Full Context

Reads your entire project — files, structure, history. Understands beyond a single file or cursor position.

→

Autonomous Agent

Executes shell commands, edits files, runs tests, installs deps, plans multi-step changes — all on its own.

⚙

Permission-Aware

Every action gated behind configurable policies. Safe for production — you stay in control.

⊕

Pluggable

MCP server support, custom skills, plugin system. Integrate with your existing tools and workflows.

◎

Web UI

Launch a full web interface with one command. REST + WebSocket API for remote sessions.

⚡ Quick Install

Get started in one line

Choose your platform — the binary is signed and verified after download.

# Linux & macOS (curl)
curl -fsSL https://raw.githubusercontent.com/calenard/calen-code/main/install.sh | sh

# Or with wget
wget -qO- https://raw.githubusercontent.com/calenard/calen-code/main/install.sh | sh
# Termux (Android)
pkg add curl
curl -fsSL https://raw.githubusercontent.com/calenard/calen-code/main/install.sh | sh

# Add to PATH:
export PATH="$HOME/.local/bin:$PATH"
# Windows PowerShell (as Admin)
irm https://raw.githubusercontent.com/calenard/calen-code/main/install.ps1 | iex

Installed to ~/.local/bin/calen  ·  Run calen --help
