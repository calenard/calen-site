✦ Capabilities

Everything a terminal agent should do

From reading your codebase to shipping the final binary — Calen Code works with you, not against you.

⌘

Terminal-Native

No Electron, no bloat. A single ~26 MB binary with zero runtime dependencies. Works on Linux, macOS, Windows, and Termux (Android).

⚡

Full Codebase Awareness

Reads your entire project — files, structure, history. Understands context beyond a single file or cursor position.

🛠

Autonomous Agent

Executes shell commands, edits files, runs tests, installs dependencies, and plans multi-step changes — all on its own.

🔒

Permission-Aware

Every action can be gated behind configurable permission policies. Safe for production — you stay in control.

📦

Pluggable & Extensible

MCP server support, custom skills, and a rich plugin system. Integrate with your existing tools and workflows.

🌐

Web UI Included

Launch a full web interface with one command. REST + WebSocket API for remote sessions and custom frontends.

⚡ Quick Install

Get started in one line

Choose your platform — the binary is signed and verified after download.

# Linux & macOS (curl)
curl -fsSL https://raw.githubusercontent.com/calenard/calen-code/main/install.sh | sh

# Or with wget
wget -qO- https://raw.githubusercontent.com/calenard/calen-code/main/install.sh | sh
# Termux (Android) — same script
pkg add curl
curl -fsSL https://raw.githubusercontent.com/calenard/calen-code/main/install.sh | sh

# Binary is installed to ~/.local/bin/calen
# Make sure it's in your PATH:
export PATH="$HOME/.local/bin:$PATH"
# Windows PowerShell (run as Administrator)
irm https://raw.githubusercontent.com/calenard/calen-code/main/install.ps1 | iex

Installs to ~/.local/bin/calen  ·  Run calen --help to get started
